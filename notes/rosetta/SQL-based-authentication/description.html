This task has three parts:

  • Connect to a MySQL database (connect_db)
  • Create user/password records in the following table (create_user)
  • Authenticate login requests against the table (authenticateuser) This is the table definition: create table users ( userid int primary key autoincrement
    username varchar(32) unique key not null
    passsalt tinyblob not null
    -- a string of 16 random bytes pass    md5 tinyblob not null -- binary MD5 hash of passsalt concatenated with the password ); (passsalt and pass_md5 would be binary(16) values, but MySQL versions before 5.0.15 strip trailing spaces when selecting them.)
