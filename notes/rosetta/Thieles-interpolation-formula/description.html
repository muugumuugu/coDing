wp:Thiele'sinterpolationformula|Thiele's interpolation formula]">[continued fraction">wp:Thiele'sinterpolationformula|Thiele's interpolation formula] is an interpolation formula for a function f(•) of a single variable.   It is expressed as a [[continued fraction: :: f(x) = f(x1) + \cfrac{x-x1}{\rho1(x1,x2) + \cfrac{x-x2}{\rho2(x1,x2,x3) - f(x1) + \cfrac{x-x3}{\rho3(x1,x2,x3,x4) - \rho1(x1,x2) + \cdots}}} \rho   represents the   wp:reciprocal difference|reciprocal difference,   demonstrated here for reference: :: \rho1(x0, x1) = \frac{x0 - x1}{f(x0) - f(x1)} :: \rho2(x0, x1, x2) = \frac{x0 - x2}{\rho1(x0, x1) - \rho1(x1, x2)} + f(x1) :: \rhon(x0,x1,\ldots,xn)=\frac{x0-xn}{\rho{n-1}(x0,x1,\ldots,x{n-1})-\rho{n-1}(x1,x2,\ldots,xn)}+\rho{n-2}(x1,\ldots,x_{n-1}) Demonstrate Thiele's interpolation function by:
