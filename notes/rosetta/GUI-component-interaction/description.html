    or increment its value with the "increment" button. Entering a non-numeric value should be either impossible
    or issue an error message. Pressing the "random" button presents a confirmation dialog
    and resets the field's value to a random value if the answer is "Yes". (This task may be regarded as an extension of the task Simple windowed application).
