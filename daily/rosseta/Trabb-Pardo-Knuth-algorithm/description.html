The TPK algorithm is an early example of a programming chrestomathy. It was used in Donald Knuth and Luis Trabb Pardo's Stanford tech report http://bitsavers.org/pdf/stanford/cstechReports/STAN-CS-76-562EarlyDevelPgmgLang_Aug76.pdf The Early Development of Programming Languages. The report traces the early history of work in developing computer languages in the 1940s and 1950s, giving several translations of the algorithm.
