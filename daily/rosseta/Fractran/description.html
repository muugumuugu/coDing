wp:FRACTRAN|FRACTRAN]">[wp:John Horton Conway|John Horton Conway">wp:FRACTRAN|FRACTRAN] is a Turing-complete esoteric programming language invented by the mathematician [[wp:John Horton Conway|John Horton Conway. A FRACTRAN program is an ordered list of positive fractions P = (f1, f2, \ldots, f_m), together with an initial positive integer input n. The program is run by updating the integer n as follows:

  • for the first fraction, fi, in the list for which nfi is an integer, replace n with nf_i ;
  • repeat this rule until no fraction in the list produces an integer when multiplied by n, then halt.
    Conway gave a program for primes in FRACTRAN: : 17/91, 78/85, 19/51, 23/38, 29/33, 77/29, 95/23, 77/19, 1/17, 11/13, 13/11, 15/14, 15/2, 55/1 Starting with n=2, this FRACTRAN program will change n to 15=2\times (15/2), then 825=15\times (55/1), generating the following sequence of integers: : 2, 15, 825, 725, 1925, 2275, 425, 390, 330, 290, 770, \ldots After 2, this sequence contains the following powers of 2: :2^2=4, 2^3=8, 2^5=32, 2^7=128, 2^{11}=2048, 2^{13}=8192, 2^{17}=131072, 2^{19}=524288, \ldots which are the prime powers of 2.
    Task: Write a program that reads a list of fractions in a natural format from the keyboard or from a string
    to parse it into a sequence of fractions (i.e. two integers)
    and runs the FRACTRAN starting from a provided integer, writing the result at each step. It is also required that the number of steps is limited (by a parameter easy to find).
    Extra credit: Use this program to derive the first 20 or so prime numbers.
