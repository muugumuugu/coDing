These number series are an expansion of the ordinary Fibonacci sequence where:

For n = 2 we have the Fibonacci sequence; with initial values 1, 1 and Fk^2 = F{k-1}^2 + F_{k-2}^2

For n = 3 we have the tribonacci sequence; with initial values 1, 1, 2 and Fk^3 = F{k-1}^3 + F{k-2}^3 + F{k-3}^3

For n = 4 we have the tetranacci sequence; with initial values 1, 1, 2, 4 and Fk^4 = F{k-1}^4 + F{k-2}^4 + F{k-3}^4 + F_{k-4}^4
...

For general n>2 we have the Fibonacci n-step sequence - Fk^n; with initial values of the first n values of the (n-1)'th Fibonacci n-step sequence Fk^{n-1}; and k'th value of this n'th sequence being Fk^n = \sum{i=1}^{(n)} {F_{k-i}^{(n)}}
