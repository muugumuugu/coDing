The http://www.cse.iitk.ac.in/users/manindra/algebra/primality_v6.pdf AKS algorithm for testing whether a number is prime is a polynomial-time algorithm based on an elementary theorem about Pascal triangles. The theorem on which the test is based can be stated as follows:

  •   a number   p   is prime   if and only if   all the coefficients of the polynomial expansion of ::: (x-1)^p - (x^p - 1) are divisible by   p.
    Example: Using   p=3: (x-1)^3 - (x^3 - 1) = (x^3 - 3x^2 + 3x - 1) - (x^3 - 1) = -3x^2 + 3x And all the coefficients are divisible by 3',   so '3 is prime. {{alertbox|#ffe4e4|Note:'
    This task is '    not     the AKS primality test.   It is an inefficient exponential time algorithm discovered in the late 1600s and used as an introductory lemma in the AKS derivation.}}

Create a function/subroutine/method that given   p   generates the coefficients of the expanded polynomial representation of   (x-1)^p.
