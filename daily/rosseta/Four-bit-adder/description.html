Task: "Simulate" a four-bit adder. This design can be realized using four wp:Adder(electronics)#Fulladder|1-bit full adders. Each of these 1-bit full adders can be built with two wp:Adder(electronics)#Halfadder|half adder]">[wp:Logic gate|gate">wp:Adder(electronics)#Halfadder|half adder]s and an or [[wp:Logic gate|gate. Finally a half adder can be made using a xor gate and an and gate. The xor gate can be made using two nots, two ands and one or. Not', 'or' and 'and, the only allowed "gates" for the task, can be "imitated" by using the Bitwise operations|bitwise operators of your language. If there is not a bit type in your language, to be sure that the not does not "invert" all the other bits of the basic type (e.g. a byte) we are not interested in, you can use an extra nand (and then not) with the constant 1 on one input.
