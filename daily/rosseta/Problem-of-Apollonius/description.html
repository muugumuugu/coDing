Task: Implement a solution to the Problem of Apollonius   (wp:ProblemofApollonius|description on Wikipedia]">[wp:ProblemofApollonius#Algebraicsolutions|algebraic solution">wp:ProblemofApollonius|description on Wikipedia])   which is the problem of finding the circle that is tangent to three specified circles.   There is an [[wp:ProblemofApollonius#Algebraicsolutions|algebraic solution which is pretty straightforward. The solutions to the example in the code are shown in the image (below and right).   The red circle is "internally tangent" to all three black circles, and the green circle is "externally tangent" to all three black circles.
