Task: Write a program which counts up from   1,   displaying each number as the multiplication of its prime factors. For the purpose of this task,   1   (unity)   may be shown as itself.
Example:       2   is prime,   so it would be shown as itself.
      6'   is not prime;   it would be shown as   '2\times3.
2144'   is not prime;   it would be shown as   '2\times2\times2\times2\times2\times67.
